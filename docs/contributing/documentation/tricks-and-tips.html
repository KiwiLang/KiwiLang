Kiwi Book

Tricks and Tips

It's a collection of useful tricks, tips and tutorials. The list updates regularly as we find new tricks and tips.

You can also contribute to this list by adding your own tricks and tips, just contact us, and we'll add it to the list.

Links to other pages

Links can be relative to the current page or absolute.

[link-text](/absolute-path/to-file.html)
[link-text](relative-path/to-file.md)

You can also use # to specify heading id.

[link-text](/absolute-path/to-file.html#heading-id)
[link-text](relative-path/to-file.md#heading-id)

Note

Absolute path starts with / and can't refer to md files, instead use .html postfix.

Using admonitions

You can use admonitions to present information in a more appealing way using admonish preprocessor.

Use the following syntax to create admonitions:

```admonish <type> [title="<title>"] [class="<class>"]
<content>
```

See example below:

```admonish warning
This is a note.

It can contain multiple lines.

<strong>
    Or even contain html tags or any other markdown features.
</strong>
```

Warning

This is a note.

It can contain multiple lines.

Or even contain html tags or any other markdown features.

See the list of available admonitions here.

Using emojis

You can use unicode shortcodes to insert emojis instead of searching for them on the internet.

It's possible by using emojicodes preprocessor, which uses gemoji assets. Use the following syntax to insert emojis:

:<shortcode>:

Wide page mode

If you want to make a page wider by removing right sidebar with navigation, you can put special comment at the top of the page.

<!--- wide page --->

In this mode, the page will be wider and will not have right sidebar with navigation.

Inserting themed SVG

You can insert svg images that will change color depending on the theme of the website.

To do so, we are using template preprocessor.

<span class="colored-fill">
{{ #template <path-to-svg> }}
</span>

See example below:

<span class="colored-fill">
{{ #template ../../assets/rocket.svg }}
</span>

All svg images are stored in assets folder of documentation source directory, and before using them, make sure to delete color attributes from svg files.

Note

Note that the path is relative to the current file.
